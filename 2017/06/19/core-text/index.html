core_text

发表于 2017-06-19

Core Text: 是一个低级文本引擎，基于Core Graphics。
Core Graphics/Quartz framework：提供对布局和格式的细粒度控制。

开始

  1. 新建 Swit项目：CoreTextMagazine
  2. 导入 CoreText.framework
    -> Linked Frameworks and Libraries -> CoreText.framework
  3. 新建 CTView.Swift 集成 UIView
  4. 修改 StoryBoard 中 View 对应的类为：CTView

显示结果

很明显：文字在左下角，而且文字是显示上下是反的。

解决

主线程检查器
user defined runtime attributes
