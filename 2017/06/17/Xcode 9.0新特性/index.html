xcode9.0 新特性

发表于 2017-06-17

模拟器多开、无线调试、标点符号自动补全

模拟器多开

想运行几个运行几个，就是那么任性
从左到右依次是 se / 6 / 7 / 7plus

无线调试

再也不用连数据线调试了，公司很多粘了很多层胶带还在服役的数据线终于可以退役了
操作步骤：

  1. 手机连接到mac， Xcode中出现连接的手机
  2. window -> devices and simulators -> devices
  3. 选中连接的手机 -> 勾选 Connect via network
  4. 等待mac与手机配对
  5. 拔掉数据线

没有 Connect via network 选项

iPhone 先升级到 iOS11，才会出现。

标点自动补全

选择一段文本，在开头输入分隔符将在最后添加一个对应的关闭分隔符。 自动匹配的分隔符包括“”，（），[]，{}
