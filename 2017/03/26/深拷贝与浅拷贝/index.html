浅拷贝与深拷贝

发表于 2017-03-26

深复制浅复制本质区别

第一部分

深复制浅复制本质区别：

  • 浅复制，源对象，新对象互相影响（修改新对象内容，源对象内容也会发生改变）
  • 深复制，源对象，新对象互不影响 (修改新对象内容，源对象内容不会发生改变)

第二部分

深复制有两种方式：copy、mutablecopy。

copy：返回一个不可变副本
mutableCopy:返回一个可变副本

深复制结果什么情况下不会生成新对象，什么情况下会生成新对象？

  • 不可变对象 copy NO
  • 不可变对象 mutablecopy YES
  • 可变对象 copy YES
  • 可变对象 mutablecopy YES
