使用中括号（[]）创建数组和字典，并通过在中括号中写入 index 或 key 来访问它们的元素。最后一个元素后允许添加逗号。

流程控制

使用if和switch来创建条件，并使用for-in，for，while和repeat-while来创建循环。 条件或循环变量周围的括号是可选的。 body周围的大括号是必须的。

在一个if语句中，条件必须是一个布尔表达式 - 这意味着代码，如分数{…}是一个错误，而不是一个隐含的比较为零。

您可以使用if和let一起处理可能丢失的值。 这些值表示为可选项。 一个可选值包含一个值，或者包含一个nil，表示一个值丢失。 在一个值的类型之后写一个问号（？），以将值标记为可选。

如果可选值为nil，则条件为false，并且跳过大括号中的代码。 否则，可选值被解包并分配给let之后的常量，这使得代码块内的解包值可用。

处理可选值的另一种方法是使用?? 操作符。 如果缺少可选值，则使用默认值。

Switches 支持任何类型的数据和各种比较操作 - 它们不限于整数和相等测试。
