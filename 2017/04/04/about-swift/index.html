about swift

发表于 2017-04-04

Swift 是针对 iOS，macOS，watchOS 和 tvOS 应用程序的全新编程语言，构建于 C 和 Objective-C 优点之上，而不受C兼容性的限制。Swift 采用安全的编程模式，并添加了现代功能，使编程更容易，更灵活，更有趣。Swift 是十分简洁的，由成熟和备受喜爱的 Cocoa 和 Cocoa Touch 框架提供支持，给重新定义软件开发方式提供了一个机会。

Swift 已经问世多年了。苹果通过推进现有的编译器，调试器和框架基础设施为Swift打下了基础。 我们通过自动参考计数（ARC）简化了内存管理。 我们的框架栈，建立在 Foundation 和Cocoa 的坚实基础上，已经现代化和标准化。 Objective-C 本身已经演变为支持 blocks, collection literals, and modules，使框架能够采用现代语言技术，而不会中断。 由于这个基础，我们现在可以为Apple软件开发的未来介绍一种新的语言。

Objective-C 开发人员 对 swift 感觉熟悉。 因为它采用了 Objective-C 命名参数的可读性和 Objective-C 动态对象模型的强大功能。 它提供对现有 Cocoa 框架的无缝访问，并与 Objective-C 代码进行混合和匹配调用。 从这个共同点构建，Swift引入了许多新功能，统一了语言的程序和面向对象部分。

Swift 对新程序员很友好。 它是第一个既满足工业标准又像脚本语言一样充满表现力和趣味的系统编程语言。 它支持 playgrounds，这是一个创新功能，允许程序员尝试 Swift 代码并立即查看结果，而无需 build 和 run app。

Swift 将最好的现代语言思想与更广泛的苹果工程师文化的智慧相结合。 编译器针对性能进行了优化，并且语言针对开发进行了优化，而不会妥协。 它旨在从 “hello, world” 扩展到整个操作系统。 所有这一切使Swift成为开发者和苹果公司对未来的投资。

Swift是编写iOS，macOS，watchOS和tvOS应用程序的绝佳方式，并将继续发展新的功能和功能。 我们对Swift的目标是雄心勃勃的。 我们迫不及待想看看您创建的内容。
