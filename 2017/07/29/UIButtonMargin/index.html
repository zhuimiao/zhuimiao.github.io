代码解释

上面4个方法，设置内部间距太简单，不解释。就挑：调整 title 距离左边框间距解释吧。

  1. 获取 titleLabel 的宽度：titleW
  2. 获取 image 宽度：imgW
  3. 获取 titleLabel 宽度 + image 宽度：totalWidth
  4. 获取 titleLabel 的默认左边框距离 （UIButton 宽度 - totalWidth）/2 ： half
  5. 获取 titleLabel 实际左边框距离 leftMargin - half : margin (如果leftMargin < half 即margin 是负数，往左移动；如果如果leftMargin > half 即margin 是正数，往右移动)
  6. 调整 titleLable 的内间距 [self setTitleEdgeInsets:UIEdgeInsetsMake(0, margin, 0, -margin)];（左间距和右间距要一起调整才会有效果。）
