常用展示方式

思路：

  1. 设置布局 （UICollectionView 的 上下左右内间距）
  2. 设置布局 cell 间的距离 （不用以为一行cell就是一个section，完全可以多行只是一个section）
  3. 设置布局 cell 的 size
  4. 设置代理方法： 返回多少个cell， 返回cell

瀑布流

瀑布流展示需要自定义 UICollectionViewFlowLayout
实现思路：

  1. 设置布局 （UICollectionView 的 上下左右内间距）
  2. 设置布局 cell 间的距离 （不用以为一行cell就是一个section，完全可以多行只是一个section）
  3. 设置布局 一共有几列 （到这里就可以计算出 cell 的宽度）
  4. 获取所有 cell 的高度：（这个容易算，cell 宽度固定，真实高度等比例）
  5. 调整所有 cell 的frame: （添加cell 的顺序是固定的，根据高低调整item的水平位置）
    推荐参考这篇博客：http://www.jianshu.com/p/126bfc28c400

分组头视图

实现步骤：

  1. collectionView 注册 registerClass: forSupplementaryViewOfKind: withReuseIdentifier:
  2. 代理方法返回size：collectionView: layout: referenceSizeForHeaderInSection:
  3. 代理方法返回view：collectionView: viewForSupplementaryElementOfKind: atIndexPath:

tips: 这里有个坑，如果设置了size，没有返回分组头 view 会崩溃。
